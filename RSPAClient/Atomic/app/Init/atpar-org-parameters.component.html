Administration > Org Parameters


Parameter Value Description
Allocation of Carts to Users Required [Unchecking this will allow any user to download Any Cart. Checking this will restrict the HHT user download only the carts allocated to the user]
Create Requisitions by Par Location for Non Stock Items [Checking this will create Requisitions by Par Location for Non Stock Items instead of creating requisition by Vendor for a given Par Location]
Default Manufacturer Item ID
[Sets the Manufacturer Item ID to the selected option]
Display of last order quantity for kill type items [Checking this will display the last order quantity and last order date for Kill type of items on HHT]
Display of Soft Input Panel (Keypad) for numeric [Checking this will automatically display the Soft Input Panel (Keypad) for numeric on Item details screen]
Display of Soft Input Panel (Keypad) [Checking this will automatically display the Soft Input Panel (Keypad) on all screens except Item details]
Factor of safety percentage (%) for recommended par [Recommended Par will be calculated based on the Factor of safety, Recommended Par = Round (Average Usage * (1+Factor of Safety/100) ) ]
Create HL7 billing message/file [Checking this will send a HL7 billing message, unchecking will create a HL7 billing file in upload path.]
Item Description Option on HHT


[Selected description value will be displayed]
Item Price Option on HHT


[Selected Item Price will be displayed in the reports on the server]
Use Packaging String for Labels [Checking this will allow users to utilized Packaging String for labels]
Display of All option in Records per Page drop down [Checking this will display all option in Records per Page drop down list. Unchecking this option and setting 0 (zero) in My Profile - Records Per Page will default to 10, as there is no ALL option.]
Truncate leading zeros [Checking this will truncate the leading zeros of a scanned Item ID]

Parameter Value Description
Allow users to do only put away of cart items [Unchecking this will enable the user to do counting and put away. Checking this will restrict the user to do only put away. If parameter "put away of cart items" is unchecked, this parameter does not have any value.]
Allow Qty. greater than Par Value [Checking this will allow users to enter the count quantity greater than par value]
Allow access to Carts allocated to Group Users [Checking this will allow complete access to carts allocated to other users in the group, the activity information still is captured for the user who does the actual count]
Disable bulk label printing [Checking this will disable the bulk label printing option]
Ignore modified Cart Definitions on Upload [Checking this will ignore errors on modified Cart Definitions while loading the counts into PeopleSoft Interface tables]
Charge Capture [Checking this will allow the users to Capture Charges on the handheld from Patient Charge Cards]
Default Input [Checking this will default the value to "0" in case of Count Option and to Par Value in case of Request Option on the HHT]
Select Default Printer Friendly Name [Selected printer friendly name for the default printer that will be used for label printing]
Delete Cart Items after upload [Checking this will delete the cart profile (items data) on the HHT after the counts are uploaded to the server]
Display FOQ / Max / Par value [Checking this will display FOQ or Max or Par value of the item, Un checking will allow the user to do Blind Counting on the item screen]
Edit par value [Checking this will allow handheld users to edit the Par value on the HHT and the Server users to edit the Par value in the Optimization Report. Audit trail is maintained for all the changes]
Ignore Putaway qty [Checking this will allow the Putaway Qty greater than Order Qty.Unchecking this will enable the system to generate an error when putaway Qty is greater than Order Qty.]
Item Count High% [If Item Order % (Qty Ordered / Par Value) value greater than this % value , the transaction will be captured for reporting]
Item Count Low% [If Item Order % (Qty Ordered / Par Value) value lower than this % value , the transaction will be captured for reporting]
Type Code for Item NDC [Type Code used for downloading the Item NDC from PeopleSoft]
Type Code for Item UPN [Type Code used for downloading the Item UPN from PeopleSoft]
Select a printer [Checking this will allow users to select an alternate printer for printing labels on the HHT]
Mandatory Count [Checking this will make it mandatory for the HHT user to enter Counts for all the items in the Cart / Par Location]
Print Cart header and items barcode [Checking this will enable printing Cart Header and Items barcode labels from Status screen after user confirmation]
Putaway of Cart Items [Checking this will allow the users to do put away of cart items.]
Default Count Option (Count/Request)

[Checking this will allow users to select an alternate printer for printing labels on the HHT]
Ability to Select count option on HHT [Checking this will enable the user to select the Count Option at the time of counting on the handheld]
Max. allowable Count/Request Quantity [Checking this will allow the users to do put away of cart items.]
Max. allowable number of digits [Force user to count/request qty less than number of digits set(this includes numbers and decimal) ]
To default all the counts [Checking this parameter will allow the user to default count all the items]
Silent Scan [Checking this will mute the scan beep of (Symbol) device(s) ]
Sort By Column [Checking this will allow the user to sort the columns on the HHT]
Super User [Allows user to view, modify, upload, delete other users data on handheld]
